# Origem

Clean folder for tests or local runs of Human Social.

To use:

1. Place here a `.txt` file with the original text, briefing, or caption.
2. Place one or more images `.png`, `.jpg`, `.jpeg`, or `.webp`.
3. Run the `/desdobrar` flow pointing at this folder.

The output will be created in `desdobramento/` during execution. The simplest folder to open and deliver is `desdobramento/output/`, with Feed, Stories, LinkedIn, and the final PDF organized without technical logs.
