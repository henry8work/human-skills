# internal/

Pasta de trabalho do Claude. Aqui ficam todos os arquivos de dados da campanha — briefs, descrições, referências, logs e feedback.

**O usuário não precisa abrir nada aqui.** O que interessa para ele está na pasta `output/` (resultados numerados). Quem mantém esta pasta é o Claude.

Conteúdo:
- `HANDOFF.md` — brief e estado da campanha
- `model-descriptions.md` / `product-description.md` / `env-descriptors.md` / `visual-references.md` — os quatro pilares de referência
- `ref-ids.md` — UUIDs das imagens enviadas ao Higgsfield
- `prompt-log.md` — todo prompt disparado
- `seedance-failures.md` — falhas e gatilhos de filtro
- `feedback.md` — feedback do usuário sobre cada resultado
- `model ref/` `product ref/` `environment ref/` `visual ref/` — imagens de referência originais
