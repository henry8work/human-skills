# visual ref/

Pasta interna — **o Claude salva aqui** as imagens de **referência visual** (estilo, look, mood, estética).

Referência visual é diferente de modelo/produto/ambiente: é uma imagem que mostra *o clima* que o usuário quer — paleta de cor, luz, composição, textura de filme, energia.

Você não mexe nesta pasta. Mande a imagem **no chat do Claude** e diga que é uma referência visual/de estilo. O Claude olha a imagem, transcreve em texto o que ela transmite (`visual-references.md`) e usa essa descrição no prompt na hora de gerar — para trazer algo parecido.
