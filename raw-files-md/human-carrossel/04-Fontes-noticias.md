## How this page is used

R1 reads this page on **every run** (no cache). It uses the native `web_search` and `web_fetch` tools to sweep each source — it doesn't matter whether the source has RSS, an API or just a site. R1 runs in the Remote sandbox, which blocks direct HTTP; that's why the single strategy is native search + fetch.

The `Language` column helps R1 know whether translation is needed. The `Priority` column drives the order and the weight of each source.

You can edit this page at any time. Add a row, remove one, adjust priority. The next R1 run will use it.

## Topics that matter

Include when it falls into one of these categories:

- **New model or version launch** (Sora 3, Veo 4, Claude Opus 4.7, new image model)
- **New technical capability** that changes workflow (camera control in video, lip sync, semantic editing)
- **Real, well-executed use case** by a recognized studio, agency, creator
- **Workflow or method** documented by someone who actually knows
- **Pricing, policy, availability change** that affects creatives
- **Cultural phenomenon with an AI angle** (use by a celebrity, brand, event)

---

## Topics to IGNORE

R1 discards:

- Twitter hot takes without a primary source
- Community drama, personality controversies
- Funding rounds and M&A without product change
- "X startup raised Y million" pure
- Opinion about the future of AI without a concrete case
- Regulation news with no direct effect on a tool
- "10 amazing prompts for you to use"
- News already covered in the last 72h (dedup by URL + title)

---

## Relevance criterion

Each news item gets a 1-5 score:

| Score | Criterion |
|---|---|
| **5** | Major model/tool launch + relevant new capability + primary source. Ex: "OpenAI launches Sora 3 with dolly camera control." |
| **4** | Launch or new capability, but with more limited scope or from a mid-tier player. Ex: "Higgsfield launches Soul 2 with 4 new trends." |
| **3** | Strong editorial analysis from a respected source about a verifiable trend. Ex: "Stratechery — why AI video killed stock production." |
| **2** | Isolated use case, interesting but not unprecedented. Ex: "São Paulo studio used Veo 3 in a Skol commercial." |
| **1** | Marginally interesting. Ex: "List of the 10 most popular AI tools in October." |

R2 only considers news items with **relevance ≥ 3**.

## Inclusion criterion

When in doubt between including or discarding a new source, ask:

> *Does this source cover {brand_subject} regularly (at least weekly) and with defensible editorial quality?*

If yes → include. If no → leave it out.

A lean feed > a feed full of noise.
