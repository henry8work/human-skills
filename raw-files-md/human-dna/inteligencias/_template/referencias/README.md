# referencias/ — DNA inputs

Place here all the material that should feed the Creative DNA before the briefing begins.

Anything that helps understand the brand counts:

- logos, symbols, monograms, signatures and variations;
- images, photographs, screenshots, moodboards and visual references;
- palettes, colors in hex, combination examples;
- fonts, typography names or `.ttf`, `.otf`, `.woff` files;
- texts, captions, emails, pages, manifestos and tone examples;
- decks, PDFs, old manuals and strategy documents;
- competitors and anti-references;
- links saved in `.txt` or `.md`.

## References for deep analysis

If you already have a brand/campaign live, place everything that can feed the DNA here. You can create a `links.md` file with public links and also drop loose files or in subfolders.

Useful links:

- main site and important internal pages;
- Instagram, LinkedIn, TikTok, YouTube, Pinterest and other networks;
- portfolio, Behance, Dribbble, Cargo, cases, public Notion;
- blog, newsletter, articles, manifestos;
- landing pages, campaign pages, sales pages;
- decks, PDFs, press kit, media kit;
- founder/team, interviews, talks, podcasts;
- press, reviews and external mentions;
- direct competitors;
- aspirational references;
- anti-references.

Useful files:

- photos, stills, frames, screenshots and moodboards;
- videos, reels, VFX, motion references and selfie videos;
- PDFs, decks, proposals, press kit and old manuals;
- mockups, applications, ads, presentations and old pieces;
- captions, emails, manifestos, scripts and loose texts;
- fonts, logos, palettes and identity files.

If there is a logo, it's one of the most important materials. Send the best version available, preferably in SVG, PDF, large PNG or vector file. If you only have the logo inside a website, screenshot, PDF or mockup, send it anyway. The Maestro tries to extract it and uses it in the `DNA.pdf` when quality permits.

Recommended structure:

```text
referencias/
├── links.md
├── 01-links-proprios/
├── 02-arquivos-visuais/
├── 03-video-motion-vfx/
├── 04-documentos-pdfs-decks/
├── 05-textos-voz/
├── 06-concorrentes/
├── 07-referencias-aspiracionais/
└── 08-anti-referencias/
```

The more real material, the more precise the DNA gets. The Maestro can also ask for these references in conversation, receive everything mixed, organize by category and ask if there's anything else until you confirm you've finished sending all references.

The Maestro will read this folder before starting generation. If there are images, they must be analyzed in `resultado/DNA.md` and compose the `resultado/DNA.pdf`.
