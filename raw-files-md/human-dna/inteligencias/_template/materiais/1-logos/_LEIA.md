# 1 — Logos

Drop any logo file here. PNG, SVG, JPG, PDF — any format works.

**If you have multiple versions** (horizontal, vertical, isolated symbol, monochrome), drop them all. You can use simple names like `logo-main.png`, `logo-vertical.png`, `symbol.png` — but don't stress, I'll look and understand.

**If you don't have a logo yet**, leave this folder empty. I'll use the brand name in a cover font until you have one.

**Tip:** if you have SVG or PNG with transparent background, even better — it looks cleaner in the pieces.
