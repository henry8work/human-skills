# campaigns/

Cada campanha (projeto) vive numa pasta isolada aqui dentro — com seus próprios refs, UUIDs, descritores, logs e outputs. Campanhas não se misturam.

**Não crie pastas aqui na mão.** Diga ao Claude "nova campanha" e ele copia o `_template/`, dá o nome certo e preenche os arquivos. Veja `COMECE-AQUI.md` → Parte 1.

Para alternar entre campanhas, diga "trocar de campanha".
