# 2 — Typography

Drop font files here (`.ttf`, `.otf`, `.woff`, `.woff2`) that you want to use for your brand.

**You may not have a font yet.** If you don't, leave it empty — during the conversation I'll help you choose (I'll show you 3-4 free options that match your brand's style).

**If you already have a defined font** (e.g. bought a font, or use a Google Fonts font):
- If it's from Google Fonts, just tell me the name — no need to download
- If it's a font you bought or have the file for, drop it here

**Common types it makes sense to have:**
- A font for **large titles** (headlines, covers, hero) — usually something with personality
- A font for **body text** (paragraphs, captions) — usually something legible and neutral
- Optionally a **mono** font (tags, metadata) — optional

No stress if only one font comes in. I'll suggest the rest.
